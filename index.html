💡 רוצה שהמערכת תעבוד מכל מקום?

העלה את הקובץ ל-GitHub Pages והמערכת תהיה זמינה מכל מחשב!

📖 מדריך GitHub Pages

✨ מה תקבל:

  • 🌐 כתובת אינטרנט קבועה - גישה מכל מקום
  • 🔥 Firebase יעבוד מושלם - סנכרון ענן מלא
  • 👥 שיתוף קל - תן קישור למורים אחרים
  • 💾 גיבוי אוטומטי - GitHub שומר הכל
  • 🆓 חינמי לחלוטין!

📋 השלבים (10 דקות):

שלב 1: הכנה

  1. שמור את הקוד הזה בקובץ בשם index.html
  2. חשוב! בדיוק השם הזה - index.html

שלב 2: GitHub

  1. היכנס ל-github.com
  2. צור חשבון חינמי (אם אין לך)
  3. לחץ על כפתור "New" הירוק
  4. שם לפרויקט: computer-system
  5. וודא ש-"Public" מסומן
  6. לחץ "Create repository"

שלב 3: העלאה

  1. לחץ "uploading an existing file"
  2. גרור את index.html לתיבה
  3. בתחתית כתוב: "Added computer system"
  4. לחץ "Commit changes"

שלב 4: הפעלת האתר

  1. לחץ על "Settings" (למעלה)
  2. בתפריט השמאלי לחץ "Pages"
  3. ב-"Source" בחר "Deploy from a branch"
  4. בחר "main" branch
  5. לחץ "Save"
  6. חכה 2-5 דקות ותקבל כתובת!

🎯 התוצאה:

תקבל כתובת כמו:

https://yourusername.github.io/computer-system
